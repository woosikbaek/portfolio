# 백우식 포트폴리오

AI & IoT 기반 스마트 솔루션 개발자의 포트폴리오 웹사이트입니다.

## 📁 프로젝트 구조

```
portfolio/
├── src/
│   ├── components/        # React 컴포넌트 모음
│   │   ├── About/         # 소개 섹션
│   │   ├── Contact/       # 연락처 섹션
│   │   ├── Footer/        # 푸터 섹션
│   │   ├── Hero/          # 메인 히어로 섹션
│   │   ├── Navbar/        # 네비게이션 바
│   │   ├── Projects/      # 프로젝트 섹션
│   │   └── Skills/        # 스킬 섹션
│   ├── styles/            # 공유 스타일
│   │   └── utilities.css  # 유틸리티 클래스
│   ├── App.jsx            # 메인 App 컴포넌트
│   ├── App.css            # 전역 스타일
│   ├── index.css          # 기본 스타일
│   └── main.jsx           # 엔트리 포인트
├── public/                # 정적 파일
├── index.html             # HTML 메인 파일
├── package.json           # 프로젝트 의존성
├── vite.config.js         # Vite 설정
├── eslint.config.js       # ESLint 설정
└── README.md              # 이 파일
```

## 🚀 빠른 시작

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```

## 🎨 스타일 시스템

### CSS Modules
각 컴포넌트는 자신의 CSS Module을 사용합니다:
- `Component.module.css` - 컴포넌트별 스타일

### 공유 스타일
- `styles/utilities.css` - 공통 유틸리티 클래스
- `App.css` - 전역 스타일
- `index.css` - 기본 리셋 및 기본 스타일

## ✨ 주요 특징

- **모던 디자인**: 글래스모피즘, 그라디언트 효과
- **부드러운 애니메이션**: CSS 애니메이션, 트랜지션
- **인터랙티브 요소**: 호버 효과, 3D 변환
- **반응형 디자인**: 모든 디바이스 지원
- **성능 최적화**: Vite 번들러 사용

## 🔧 기술 스택

- **React** - UI 라이브러리
- **Vite** - 빌드 도구
- **CSS3** - 스타일링 (CSS Modules)
- **JavaScript ES6+** - 개발 언어
- **ESLint** - 코드 품질 관리

## 📱 반응형 지원

- 모바일 (480px 이상)
- 태블릿 (768px 이상)
- 데스크톱 (1024px 이상)
- **터치 최적화**: 모바일 기기를 위한 터치 대상
- **성능 최적화**: 모바일 기기에서도 부드러운 애니메이션

## 🎯 성능 최적화

### 로딩 성능
- **지연 로딩**: 필요할 때만 애니메이션 실행
- **스크롤 최적화**: 디바운스 처리된 스크롤 이벤트
- **리소스 최적화**: 효율적인 CSS 애니메이션
- **압축**: 최소화된 코드

### 접근성
- **키보드 네비게이션**: Tab 키로 전체 사이트 탐색
- **스크린 리더**: ARIA 레이블과 시맨틱 HTML
- **색상 대비**: WCAG 2.1 기준 충족
- **모션 감소**: prefers-reduced-motion 지원
